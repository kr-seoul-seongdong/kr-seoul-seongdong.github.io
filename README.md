# 성동구 · 서울특별시

> 성동구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-seongdong.github.io/**

성동구는 서울특별시 동부에 자리한 자치구로, 한강 북쪽의 왕십리·성수 일대를 아우르는 지역입니다. 예로부터 도성 동쪽 관문이자 강남과 강북을 잇는 길목이었으며, 오늘날에는 성수동을 중심으로 한 도시 재생과 산업 전환으로 서울에서 가장 빠르게 변모하는 지역 중 하나로 꼽힙니다. 옛 수제화 공장과 인쇄소가 밀집했던 준공업지대가 개성 있는 카페와 편집숍, 스타트업 오피스로 탈바꿈하면서 젊은 층이 즐겨 찾는 명소가 되었습니다. 구의 중앙에는 서울숲이라는 대규모 도시공원이 있어 산업적 활력과 녹지 휴식이 공존하는 독특한 성격을 지닙니다. 구청은 행당동 고산자로 270에 있으며, 왕십리역을 축으로 여러 지하철 노선이 교차해 접근성이 매우 뛰어납니다. 성동구는 전통과 첨단, 생산과 여가가 한데 어우러진 서울 동부의 관문 도시입니다.
